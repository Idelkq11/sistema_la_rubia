# 🧾 Sistema de Facturación "La Rubia"

Este es un sistema web básico para registrar facturas, pensado para negocios pequeños. Permite ingresar clientes, artículos, calcular totales y ver reportes diarios. Desarrollado en PHP con base de datos MySQL como parte de la **Tarea 8**.

## Acceso al sistema

- **Usuario:** demo  
- **Clave:** tareafacil25

Este usuario se crea automáticamente cuando ejecutas `install.php`.

## Instalación paso a paso

1. Descarga o clona este repositorio.
2. Coloca la carpeta `sistema_la_rubia` dentro de `htdocs` de XAMPP.
3. Abre XAMPP y enciende Apache y MySQL.
4. En tu navegador ve a:http://localhost/sistema_la_rubia/install.php
Eso creará la base de datos.
5. Luego entra a:http://localhost/sistema_la_rubia/index.php
